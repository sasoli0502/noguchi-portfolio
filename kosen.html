PROJECT: MNCT (Age 16-21)

MIYAKONOJO NATIONAL COLLEGE OF TECHNOLOGY

都城高専。そこは、社会から隔絶された場所にある「知の熱狂のるつぼ」だった。

16歳から21歳までの5年間。普通の高校生が青春を謳歌している間、俺たちは油の匂いとハンダの煙にまみれ、電子と格闘していた。回路設計、構造解析、熱力学、そして無限のデバッグ。ここでは、論理性こそが唯一の正義であり、結果こそがすべての証明だった。

高専という特殊な環境には、変人か天才しかいなかった。自分の好きなことだけに全神経を注ぐ、純粋で狂気じみた情熱を持つ仲間たち。彼らとの日々は、まさに「修行」そのものだった。

while(still_living) {
  input = NEW_KNOWLEDGE;
  process(input);
  output = CREATION;
  if (output.isSuccess()) continue;
  else fix_bugs();
}

ここでの5年間で起きたすべての出来事を語るには、このページの容量があまりにも足りない。数え切れないほどの失敗、予測不能なバグ、そして徹夜の末に見た朝日の美しさ。それらすべてが、今の俺を形作る強固なミドルウェアとなっている。

「エンジニアリングとは、世界を計算可能な形に再構成することである」

都城の大地で学んだその鉄則は、現在、キャッシュフローという名の「数値の奔流」を掌握するための強力なエンジンとして稼働している。
